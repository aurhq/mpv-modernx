# mpv-modernx (AUR)

This is the AUR package for [ModernX](https://github.com/zydezu/ModernX), a fork of mpv-osc-modern.

## Installation

You can install this package using an AUR helper:

```bash
paru -S mpv-modernx
# or
yay -S mpv-modernx
```

## Manual Installation

```bash
git clone https://aur.archlinux.org/mpv-modernx.git
cd mpv-modernx
makepkg -si
```
